## 📖 О проекте

**MangaMaster** — это веб-приложение (SPA), которое помогает структурировать и отслеживать вашу коллекцию манги, ориентируясь на официальные российские издания.  
Вся общая база данных хранится в этом же репозитории, а ваши личные отметки (коллекция, прочитано, вишлист) сохраняются в вашем браузере через `LocalStorage`.

Проект создан для фанатов манги, которые хотят:
- вести учёт своей физической коллекции;
- отмечать прочитанные тома;
- планировать будущие покупки;
- быстро находить информацию по авторам, издателям и жанрам.

## ✨ Возможности

- **🗂️ Каталог манги** с возможностью переключения вида отображения: карточки, список, обложки.
- **🔍 Поиск и фильтрация** по авторам, издательствам и жанрам.
- **📊 Личная коллекция**:
  - добавить мангу в «Моя коллекция»;
  - отметить тома как «Прочитано»;
  - добавить в «Вишлист» (хочу купить).
- **💾 Импорт / экспорт** своей коллекции в формате JSON (перенос данных между устройствами).
- **📱 Progressive Web App (PWA)** — приложение можно установить на смартфон или компьютер как нативное.
- **🔄 Синхронизация** с общей базой данных манги, которая обновляется через репозиторий.

## 🛠 Технологии

- **HTML5**, **CSS3**, **JavaScript (ES6+)**
- **LocalStorage API** — для хранения личной коллекции
- **PWA (Service Worker, Manifest)** — для работы офлайн и установки на устройство
- **GitHub Pages** — для хостинга
- **GitHub** — для хранения общей базы данных в JSON

## 📂 Структура данных

- **База манги (общая)** — хранится в репозитории в файле `database.json`. Содержит информацию об официальных российских изданиях: названия, авторы, издатели, жанры, количество томов и т.д.
- **Личная коллекция** — хранится в `localStorage` браузера под ключом `mangaMasterCollection`. Вы можете экспортировать её в JSON и импортировать обратно.
